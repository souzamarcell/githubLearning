*** Comiti - pequenas mudanda no codigo.
  . todos os arquivos que sofreram mudança ou
  index.html - arquivo que sofreu mudança
  m - mensgen (colar mensagens curtas)

  git add .
  git commit -m "created index.html"

*** Logs
  git log
  git log --oneline (forma resumida)
  git log -n 3 (3 ultimos logs)

*** Estagios do arquivo
  1) Working directory - modificado arquivo
  2) Stage Area - add (area de commit)
  3) Repository - commit

*** Status
  git status

  HEAD - ultimo commit
  linha do tempo (master/branch)
  linha do tempo (main/branch) Dias atuais 04/Nov/2022

*** Diferenca (o que foi modificado)
  git diff

*** Restore
  git restore . (para todos)
  git restore index.html (para um arquivo)

*** Remover da 2) Stage Area - add (area de commit)
  git restore --staged arquivo.xxx
  git restore --staged . (para todos)

*** Modificar o nome 3) Repository - commit
  git commit --amend -m "created index.html e zi.txt v2.4"

*** Voltar o ultimo 3) Repository - commit
  git reset --soft HEAD~1
